github + hexo

下载Node、Git

官网下载即可
均默认安装C盘

安装教程参考csdn

安装完成后用管理员运行cmd
输入以下命令检验是否安装成功

node -v
npm -v
git -v

如图所示：

![[Pasted image 20241118211651.png|234]]
![[Pasted image 20241118211701.png|231]]
![[Pasted image 20241118211708.png|244]]

下载hexo

管理员运行cmd

设置镜像

npm config set registry https://registry.npmmirror.com

查看配置信息

npm config list

![[Pasted image 20241118215256.png|347]]

安装hexo

npm install -g hexo-cli

安装完后输入验证是否安装成功

hexo -v

如果显示未知命令输入

npx hexo -v

![[Pasted image 20241118212252.png|205]]

GitHub

创建新仓库

名称必须和用户名相同

![[Pasted image 20241118212536.png|311]]

必须Public

![[Pasted image 20241118212559.png|325]]

勾选

![[Pasted image 20241118212632.png|375]]

即可创建完成

配置ssh key

在桌面右键git bush here

ssh-keygen -t rsa -C "邮箱地址"

接下来敲四下键盘

![[Pasted image 20241118212917.png|300]]

之后进入C:\Users\用户名.ssh

用任意文本编辑器打开

![[Pasted image 20241118213027.png|109]]

全选复制

打开GitHub选择settings

![[Pasted image 20241118213125.png|137]]

选中SSH

![[Pasted image 20241118213153.png|275]]

新建

![[Pasted image 20241118213214.png|375]]

标题任意，key粘贴刚刚复制的然后创建

![[Pasted image 20241118213307.png|425]]

回到桌面打开git bash here，验证是否成功添加

ssh -T git@github.com

![[Pasted image 20241118213521.png|325]]

输入yes

![[Pasted image 20241118213547.png|350]]

本地部署

创建博客文件夹，名称随意
打开文件夹并git bash here

进行初始化

hexo init

如果初始化失败则输入

npx hexo init

如果出现：

![[Pasted image 20241118222334.png|450]]

继续输入

npm install

若出现以下错误

![[Pasted image 20241118223020.png|375]]

参考解决

https://blog.csdn.net/HANZY72/article/details/122505375

安装hexo

npx hexo install
npx hexo g
npx hexo s

出现

![[Pasted image 20241118223332.png|400]]

这个网址打开即表明成功部署

![[Pasted image 20241118223354.png|300]]

Ctrl+C停止

上线博客

打开博客文件夹（推荐记事本打开）

![[Pasted image 20241118223501.png|184]]

最底部进行修改

deploy:
(空一格)type: git
(空一格)repository: 仓库链接
(空一格)branch: main

如图

![[Pasted image 20241118230954.png|175]]

保存退出

在博客文件夹git bash here

npx hexo g

npx hexo d

如果显示以下错误

![[Pasted image 20241118224211.png|320]]

输入即可

npm install hexo-deployer-git --save

由于第一次使用会提示

![[Pasted image 20241118224751.png|425]]

分别输入

git config --global user.email "邮箱"
git config --global user.name "GitHub用户名"

弹出窗口

![[Pasted image 20241118224959.png|200]]

选择蓝色的

![[Pasted image 20241118225028.png|136]]

git bash here出现下面的就成功了

![[Pasted image 20241118225119.png|275]]

等待片刻就可以进入博客

链接就是仓库名称lxhei1831.github.io

上传

==注意关闭代理关闭梯子！！！==
右键博客目录git bash here

npx hexo clean
npx hexo g
npx hexo d

刷新即可
